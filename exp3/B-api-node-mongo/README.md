# Exp3-B APIs with Node + Express + MongoDB

This folder now has its own duplicate files:
- `server.js`
- `package.json`
- `.env.example`
- `public/index.html`

APIs:
- `POST /api/register`
- `POST /api/login`
- `GET /api/user/:id`
- `DELETE /api/user/:id`

Run:

```bash
cd exp3/B-api-node-mongo
copy .env.example .env
npm install
npm start
```
